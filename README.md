# ListPlay

ListPlay is a Chrome extension that captures songs from a playlist and adds them to a new playlist on supported music services.

## Features

- Copy all songs from an open playlist (Spotify, YouTube Music, Apple Music)
- Add captured songs to a new playlist on the current service
- Clear stored songs

## How to Use

1. Open a playlist on Spotify, YouTube Music, or Apple Music.
2. Click **Copy Playlist** to capture all songs.
3. Click **Add to Playlist** on the target playlist and add the captured songs.
4. Click **Clear** to remove stored songs.


## Notes

Keep the playlist tab open while capturing or adding songs.
